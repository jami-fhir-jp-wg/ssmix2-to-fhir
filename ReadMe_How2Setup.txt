SS-MIX2toFHIR IG セットアップ手順

１．下記のURLを参考にし、IG Publiser 環境を整えます。
  https://fshschool.org/quickstart/
 ※IG Publiser 起動前に ruby, jekll のインストールが必要なので注意しください。
 
２．Python 3 系をインストールします。
　必要なライブラリは「conceptmapgen-requirements.txt」を参照ください。

３．SuperGenonce.bat を起動します。outputにIG サイトが作成されます。
　※うまくGoogleにつながらない場合は conceptmapgenconfig.py 内のプロキシ設定を変更してください。

４．output/index.html がサイトインデックスです。
